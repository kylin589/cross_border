Amazon亚马逊店铺授权方法

填写授权信息提交授权

点击【添加Amazon授权】，按要求完成相关授权信息添加，并【添加授权】即可完成该店铺的授权。

注：Merchant ID、AWS Access Key ID和Secret Key需要登录亚马逊账号到指定页面中查找。 各站点网址如下：（为避免账号关联，请使用该店铺常用的电脑操作）

注：一般开通美国站和英国站就可以获取到美洲和欧洲的所有已开通店铺了

  • 美国（US）：https://developer.amazonservices.com
  • 英国（UK）：https://developer.amazonservices.co.uk
  • 法国（FR）：https://developer.amazonservices.fr
  • 加拿大（CA）：https://developer.amazonservices.ca
  • 德国（DE）：https://developer.amazonservices.de
  • 西班牙（ES）：https://developer.amazonservices.es
  • 印度（IN）：https://developer.amazonservices.in
  • 意大利（IT）：https://developer.amazonservices.it
  • 日本（JP）：https://developer.amazonservices.jp

以美国站点为例，打开美国站网址，点击页面上的【Sign up for MWS】，输入所授权店铺的邮箱和密码完成登录。如图：

登录后将进入如下页面，选中第一条“I want to access ……”并点击【NEXT】。

在新页面勾选中“I have read ……”并点击【NEXT】。

然后在新页面将看到“Merchant ID（部分站点为Seller ID）”、“AWS Access Key ID”、“Secret Key”，复制到后台即可。

注：将获取到的这些信息最好保存到文档中备份，以备后期需要
